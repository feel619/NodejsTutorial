# How To Install Nodejs?

* Go To [nodejs.org](https://nodejs.org/en)
* Select Nodejs LTS Version
* Download LTS Version Node.js .pkg file
* Install it like a normal application

## To Check If Node.js is installed, open Your Terminal or command prompt and type:

```
node -v
```
If you See a version number, Node.js is installed Correctly

Verify npm version:

```
npm -v
```
